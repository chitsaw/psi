    Remarks

    The component takes in a stream of text-based phrases or utterances on its In stream and detects intents and extracts any entities contained therein, posting the results on its Out stream. It works in conjunction with the Microsoft Cognitive Services LUIS API and requires a subscription key in order to use. In addition, a LUIS application needs to be created in the service ahead of time, and the application id (App Id) should be specified in the component configuration. For more information, see the complete documentation for the Microsoft Cognitive Services LUIS API.
