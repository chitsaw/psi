    Class AudioPlayer

    Component that plays back an audio stream to an output device such as the speakers.

    Remarks

    This output component renders an audio input stream of type AudioBuffer to the default or other specified audio output device for playback. The audio device on which to playback the output may be specified by name via the DeviceName configuration parameter. The GetAvailableDevices() static method may be used to enumerate the names of audio output devices currently available on the system.
    Please note: This component uses Audio APIs that are available on Windows only.
