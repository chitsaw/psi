    Defines a stream value provider, i.e. an object that provides stream values around a specified time position.

    Remarks

    A stream value provider allows stream value subscribers to register and unregister. When the ReadAndPublishStreamValue(IStreamReader, DateTime) method is called, the provider reads and publishes the stream value at a specified time to all registered subscribers. Finally, the SetCacheInterval(TimeInterval) method allows the stream value provider to be notified when the interval of interest (in which subsequent requests to read might arise) might change.
