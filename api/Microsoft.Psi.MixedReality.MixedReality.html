    Initialize(SpatialAnchor, Boolean)

    Initializes static members of the MixedReality class. Attempts to initialize the world coordinate system from a given spatial anchor. If no spatial anchor is given, it attempts to load a default spatial anchor. If the default spatial anchor is not found (e.g., if the app is being run for the first time), a stationary frame of reference for the world is created at the current location.
