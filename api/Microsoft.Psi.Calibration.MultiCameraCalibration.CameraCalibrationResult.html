    Pose

    Gets the camera's pose. The pose is used to convert from device=>world coordinates (i.e. given 0,0,0 it will tell you where the camera is located in world coordinates). Pose is essentially the inverse transformation as that defined by Extrinsics. Units are converted to meters. The matrix assumes column-vectors: (i.e. Matrix * Point versus Point * Matrix). The MathNet basis is used: Z (up) | | +----> Y (left) / / X (forward).
