    Remarks

    This class abstracts common functionality for, and is used via aggregation by PlotVisualizationObjectView<TPlotVisualizationObject, TData> and PlotSeriesVisualizationObjectView<TPlotSeriesVisualizationObject, TKey, TData>. The constructor receives the corresponding visualization object view as a parent, abstracted as a IPlotVisualizationObjectView<TKey, TData> object.
