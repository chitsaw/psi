    BufferLengthInMs

    Gets or sets the length of the recognizer's input stream buffer in milliseconds.

    Declaration
    public int BufferLengthInMs { get; set; }
    Property Value
    Type Description
    Int32
    Remarks

    Audio arriving on the input stream will be stored in an internal buffer for the speech recognition engine to read as it is able. This buffer will block when full until the recognition engine is able to read from the buffer. Set this value to modify the length of the buffer, which is computed based on the length of audio to buffer in milliseconds and the audio input format. By default, a 1000 ms buffer is used. It is safe to leave this value unchanged.

    Grammars

    Gets or sets the list of grammar files.

    Declaration
    public GrammarInfo[] Grammars { get; set; }
    Property Value
    Type Description
    GrammarInfo[]
    Remarks

    Use this to specify a set of grammar files that the recognizer should use for speech recognition. Grammar files are XML-format files that conform to the W3C Speech Recognition Grammar Specification (SRGS) Version 1.0. If this configuration property is not specified or set to null, the recognizer will use a default context-free grammar used for free text dictation.
