    CalibrateFromFileAsync(StorageFolder)

    Calibrates the camera sensor, using stored files.

    Declaration
    public async Task<bool> CalibrateFromFileAsync(StorageFolder calibrationFolder = null)
    Parameters
    Type Name Description
    Windows.Storage.StorageFolder calibrationFolder

    The device folder containing calibration files. Defaults to Documents.
    Returns
    Type Description
    Task<System.Boolean>

    True when complete.
    Remarks

    Camera intrinsics are loaded from the file: {calibrationFolder}/{ResearchModeSensorType}Intrinsics.xml, and the map of calibration points are loaded from the file: {calibrationFolder}/{ResearchModeSensorType}Points.map. Follow these steps to give your app read/write access to the Documents folder: https://docs.microsoft.com/en-us/uwp/api/windows.storage.knownfolders.documentslibrary?view=winrt-22000#prerequisites If the files do not already exist, they will be created, and calibration will be computed from scratch and saved.
