    Remarks

    The origin of the eyes and head poses are between the user's eyes. If emitting both eye and head gaze, the head poses are computed and emitted in alignment with the timestamp of each eye gaze sample. See here for more information about the WinRT APIs: https://learn.microsoft.com/en-us/windows/mixed-reality/develop/native/gaze-in-directx.
