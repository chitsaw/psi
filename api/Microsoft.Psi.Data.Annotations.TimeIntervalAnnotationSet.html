    Class TimeIntervalAnnotationSet

    Represents a set of overlapping time-interval annotations that belong to separate tracks but end at the same time.

    Inheritance
    Object
    TimeIntervalAnnotationSet
    Namespace: Microsoft.Psi.Data.Annotations
    Assembly: Microsoft.Psi.Data.dll
    Syntax
    public class TimeIntervalAnnotationSet : object
    Remarks

    This data structure provides the basis for persisting overlapping time interval annotations in \psi streams. It captures a set of overlapping time interval annotations that are on different tracks but end at the same time, captured by EndTime. When persisted to a stream, the originating time of the Message<T> should correspond to the EndTime.
