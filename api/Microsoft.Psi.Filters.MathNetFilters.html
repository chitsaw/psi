    BandpassFilter(IProducer<Double>, Double, Double, Double, ImpulseResponse, Interpolator<Double, Double>, Nullable<DateTime>, DeliveryPolicy<Double>)

    Samples the input stream and applies a band-pass filter (using MathNet.Filtering) over the sampled input stream. Removes low and high frequencies.

    Declaration
    public static IProducer<double> BandpassFilter(this IProducer<double> input, double sampleRate, double cutoffLowRate, double cutoffHighRate, ImpulseResponse impulseResponse = null, Interpolator<double, double> sampleInterpolator = null, DateTime? alignmentDateTime = null, DeliveryPolicy<double> deliveryPolicy = null)
    Parameters
    Type Name Description
    IProducer<System.Double> input

    The input source of doubles.
    System.Double sampleRate

    The desired sample rate for the filter.
    System.Double cutoffLowRate

    The desired low frequency cutoff for the filter.
    System.Double cutoffHighRate

    The desired high frequency cutoff for the filter.
    ImpulseResponse impulseResponse

    Specifies how the filter will respond to an impulse input (Finite or Infinite). Default is Finite.
    Interpolator<System.Double, System.Double> sampleInterpolator

    Optional sampling interpolator over the input. Default is a reproducible linear interpolation.
    System.Nullable<DateTime> alignmentDateTime

    If non-null, this parameter specifies a time to align the sampled messages with, and the sampled messages will have originating times that align with (i.e., are an integral number of intervals away from) the specified alignment time.
    DeliveryPolicy<System.Double> deliveryPolicy

    An optional delivery policy for the input stream.
    Returns
    Type Description

    A band-pass filtered version of the input.
