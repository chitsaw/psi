    Parameters
    Type Name Description
    Double scale

    The scale to use when calculating metric corner offsets from image pixel width and height.
    Point3D origin

    The origin of the image rectangle.
    UnitVector3D widthAxis

    The horizontal width axis of the image rectangle.
    UnitVector3D heightAxis

    The vertical height axis of the image rectangle.
    Shared<T> image

    The image.
    Remarks

    The (left, bottom) corner of the image rectangle is set to the origin (0, 0), and (width, height) are calculated from multiplying the image pixel width and height respectively by a scaling parameter. The edges of the image rectangle are aligned to the specified width and height axes.
