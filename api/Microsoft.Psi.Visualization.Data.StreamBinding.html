public class StreamBinding
    Remarks

    A stream binding contains an overall stream adapter that is formed by chaining a derived stream adapter (used to compute the values for the derived stream from the source stream) and a visualizer adapter (used to adapt the values produced by the stream to the visualizer that the binding connects to). In addition, the binding contains information about the summarizer used by the bound visualizer.
