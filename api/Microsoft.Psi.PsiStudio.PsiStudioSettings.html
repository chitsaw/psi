    Version

    Gets or sets the version of the settings object for serialization.

    Declaration
    public int Version { get; set; }
    Property Value
    Type Description
    Int32
    Remarks

    This defaults to 0 unless explicitly set in order to distinguish between older versions of the settings file (which did not have the Version attribute), and newer versions which which will contain the Version attribute set to CurrentVersion. Note that it is not necessary to bump CurrentVersion every time changes are made to PsiStudioSettings unless the serialized schema of an existing property has changed between versions and special handling is required for it. Simply adding or removing properties should not require a version change provided default values are provided for any added properties.
