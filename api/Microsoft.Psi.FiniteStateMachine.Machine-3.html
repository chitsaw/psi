    Class Machine<TInput, TOutput, TContext>

    The Machine is an abstract class expected to be implemented with app-specific methods representing inputs to the system and setting up the initial states and transitions of the system. Machine contains the current State and (for convenience) the current context. A single Update() method is expected to be called with context changes due to external inputs.
