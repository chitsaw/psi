    Remarks

    The component takes in text-based phrases or utterances on its input stream and generates textual responses, posting the results on its output stream. It works in conjunction with the Microsoft Cognitive Services Personality Chat API and requires a subscription key in order to use. For more information, see the complete documentation for the Microsoft Cognitive Services Personality Chat API.
