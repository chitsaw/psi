    Interface IPlotVisualizationObjectView<TKey, TData>

    Defines a plot visualization object view.

    Namespace: Microsoft.Psi.Visualization.Views.Visuals2D
    Assembly: Microsoft.Psi.Visualization.Windows.dll
    Syntax
    public interface IPlotVisualizationObjectView<TKey, TData>
    Type Parameters
    Name Description
    TKey

    The type of the series key.
    TData

    The type of data.
    Remarks

    This interface is used to abstract over the two different types of plot visualization object views currently available: PlotVisualizationObjectView<TPlotVisualizationObject, TData>, which is used for simple numerical plots, and PlotSeriesVisualizationObjectView<TPlotSeriesVisualizationObject, TKey, TData>, which is used for plot series. The interface enables encapsulating the common functionality for creating plot views in the PlotVisualizationObjectViewHelper<TKey, TData> class such as to avoid code duplication.
