    TargetLatencyInMs

    Gets or sets the target audio latency.

    Declaration
    public int TargetLatencyInMs { get; set; }
    Property Value
    Type Description
    Int32
    Remarks

    This parameter controls the amount of audio to resample and output at a time, which in turn determines the latency of the audio output. The larger this value, the more audio data is carried in each AudioBuffer and the longer the audio latency. For live audio capture, we normally want this value to be small as possible. By default, this value is set to 20 milliseconds. It is safe to leave this unchanged.
