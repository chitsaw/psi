    Class OnnxModel

    Implements a helper class for running ONNX models.

    Inheritance

    Namespace: Microsoft.Psi.Onnx

    Remarks

    This class implements the ability to run a specified ONNX model. It does so by leveraging the ML.NET framework. The OnnxModelConfiguration object specified at construction time provides information about where to load the network from, etc.
