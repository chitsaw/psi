    Struct AudioBuffer

    Represents a buffer of audio and its associated format information.

    Namespace: Microsoft.Psi.Audio
    Assembly: Microsoft.Psi.Audio.dll
    Syntax
    public struct AudioBuffer
    Remarks

    Audio in \psi is represented as a structure consisting of an array of bytes containing a chunk of audio data and a WaveFormat object which describes the encoding format of the audio data. While pairing each chunk of audio data with its corresponding WaveFormat object incurs some amount of overhead, it simplifies processing of audio data by stream components and operators in \psi by providing all the relevant audio information in a single unit.
