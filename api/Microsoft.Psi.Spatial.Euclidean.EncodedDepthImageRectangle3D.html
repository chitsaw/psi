    The vertical height axis of the encoded depth image rectangle.
    Shared<EncodedDepthImage> depthImage

    The encoded depth image.
    Remarks

    The (left, bottom) corner of the encoded depth image rectangle is set to the origin (0, 0), and (width, height) are calculated from multiplying the encoded depth image pixel width and height respectively by a scaling parameter. The edges of the encoded depth image rectangle are aligned to the specified width and height axes.
