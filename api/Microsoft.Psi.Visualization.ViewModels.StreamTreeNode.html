    Class StreamTreeNode

    Implements a node in the dataset tree that represents a stream.

    Syntax
    public class StreamTreeNode : ObservableTreeNodeObject
    Remarks

    Stream tree nodes can represent (1) source streams that are present in a store, (2) derived streams, i.e., streams computed from a source stream based on a derived stream adapter, or (3) stream containers (that subsume other streams).
