    Parameters
    Type Name Description
    Int32 id

    The id of the stream, unique for this store. All messages with this stream id will be written to this stream.
    String name

    The name of the stream. This name can be later used to open the stream for reading.
    System.Boolean indexed

    Indicates whether the stream is indexed or not. Indexed streams have a small index entry in the main data file and the actual message body in a large data file.
    String typeName

    A name identifying the type of the messages in this stream. This is usually a fully-qualified type name or a data contract name, but can be anything that the caller wants.
